## 앱 이름

로또될각

## 짧은 설명

번호 추천, 고정번호, 합계 생성, 통계까지 한 번에 보는 로또 도우미

## 긴 설명

로또될각은 번호 추천, 고정번호 추천, 합계 생성, 통계 분석 기능을 한 번에 제공하는 로또 도우미 앱입니다. 원하는 방식으로 번호를 만들고, 최근 데이터와 통계를 참고해 나만의 번호 조합을 편하게 확인해보세요.

주요 기능
- 번호 추천
- 고정번호 추천
- 합계 생성 추천
- 통계 분석
- 최근 당첨번호 확인

원하는 번호를 직접 고정해서 추천을 받을 수 있고, 합계 범위를 설정해 번호를 생성할 수도 있습니다. 자주 나오는 패턴과 통계를 함께 보면서 나만의 방식으로 번호를 조합해보세요.

로또될각은 복잡하지 않게, 필요한 기능만 한 번에 볼 수 있도록 구성했습니다. 가볍게 번호를 확인하고 싶을 때도, 조금 더 데이터를 참고하고 싶을 때도 편하게 사용할 수 있습니다.

## for_readme

```md
# 로또될각

데이터 기반 로또 번호 추천과 통계 분석 앱입니다.

로또될각은 과거 당첨 데이터를 바탕으로 번호 추천, 고정번호 추천, 합계 범위 추천, 통계 분석을 제공하는 로또 번호 조합 도구입니다.

## 주요 기능

- 번호추천: 최근 출현 흐름과 누적 데이터를 참고한 5세트 추천
- 고정추천: 원하는 번호를 고정하고 나머지 번호 자동 추천
- 합계생성: 원하는 합계 범위에 맞는 번호 조합 생성
- 통계: 합계, 홀짝, 연속 번호, 고빈도/저빈도 번호 확인

## 안내

이 앱은 당첨을 보장하지 않으며, 과거 데이터를 참고해 번호 선택을 돕는 보조 도구입니다.
```
