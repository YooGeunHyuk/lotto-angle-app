## 제출 전 핵심 주의사항

이 앱은 실제 로또6/45를 다루는 번호 추천/통계/당첨 확인 보조 앱이다. 직접 복권을 판매하거나 앱 안에서 베팅을 받지는 않지만, 실제 금전 복권과 관련된 정보, 예측, QR 확인, 판매점 정보를 제공한다.

따라서 App Store Connect에서 다음 항목을 보수적으로 설정한다.

- 연령 등급에서 `Gambling` 관련 항목을 반드시 `Yes` 또는 해당 빈도로 표시한다.
- 앱 설명에는 당첨을 보장한다는 표현을 쓰지 않는다.
- `1등 보장`, `당첨 확률 상승 보장`, `무조건 당첨` 같은 문구는 사용하지 않는다.
- 앱은 번호 선택을 돕는 참고용 도구이며 구매/베팅 기능이 없다는 점을 명확히 한다.
- 카메라, 위치 권한 사용 목적을 실제 기능과 일치하게 작성한다.
- 광고 SDK 사용에 따른 개인정보 라벨을 정확히 입력한다.

## Age Rating

### 권장 설정

App Store Connect > App Information > Age Rating에서 아래처럼 보수적으로 설정한다.

- Gambling: `Yes`
- Simulated Gambling: 앱 안에서 베팅/카지노형 게임을 하지 않으므로 `None` 또는 `No`
- Contests: 사용자 간 경쟁/리워드가 없으므로 `None`
- Loot Boxes: `No`
- Unrestricted Web Access: 외부 웹뷰/브라우저를 앱 내에서 자유 탐색시키지 않으면 `No`

메모:

- Apple 정의상 Gambling에는 lotteries and raffles가 포함된다.
- 이전 제출에서 이 항목 누락으로 Guideline 2.3.6 리젝 이력이 있다.
- 실제 돈을 걸거나 앱 안에서 구매하지 않더라도, 실제 로또와 관련된 정보/예측 앱이므로 `Gambling`은 숨기지 않는다.
- 결과적으로 17+ 또는 18+ 계열 등급이 나올 수 있다.
- 한국 지역에서 GRAC 관련 추가 요청이 오면 App Review 메시지에 따라 Korea age rating override를 처리한다.

## App Privacy

현재 앱 기능 기준으로 확인해야 할 데이터 유형이다.

### 직접 앱 기능

- 회원가입: 없음
- 자체 서버 로그인: 없음
- 내 번호 저장: 기기 내 AsyncStorage 저장
- QR 원문: 기기 내 저장 가능
- 위치: 주변 판매점 조회/지도 표시 용도
- 카메라: QR 스캔 용도

### 외부 SDK / 네트워크

- Google AdMob 사용
- 동행복권 공개 API/웹 엔드포인트 조회
- GitHub raw JSON 조회
- Apple Maps / Google Maps 지도 표시

### 개인정보 라벨 입력 가이드

보수적으로 검토할 항목:

- Location
  - 위치 권한을 사용한다.
  - 위치가 앱 밖 서버/외부 API로 전송되는 경우 `Precise Location` 또는 `Coarse Location` 수집 여부를 실제 구현 기준으로 확인한다.
  - 현재 판매점 조회 과정에서 위치 기반 행정구역을 추정하고 동행복권 판매점 API를 호출한다. 좌표 원문을 직접 저장하지 않더라도 위치 기반 데이터 처리가 있으므로 Privacy 입력 시 주의한다.

- Identifiers
  - AdMob이 광고 식별자 또는 기기 식별자를 사용할 수 있다.
  - IDFA/ATT 사용 여부와 AdMob 설정을 확인한다.

- Usage Data
  - AdMob 광고 노출/반응 데이터가 있을 수 있다.

- Diagnostics
  - Expo/Apple/AdMob/시스템 크래시 데이터가 수집될 수 있는지 확인한다.

### Tracking

중요:

- 개인화 광고를 사용하거나 광고 식별자를 다른 앱/웹 데이터와 연결해 광고 목적으로 사용하면 Tracking에 해당할 수 있다.
- Tracking을 한다면 ATT 권한 요청 흐름이 필요하다.
- ATT를 넣지 않을 계획이라면 AdMob을 비개인화 광고 중심으로 설정하고, 개인정보 라벨도 그에 맞춰 정리해야 한다.

권장 액션:

1. AdMob에서 개인화 광고 사용 여부 확인
2. 앱에서 ATT 요청을 넣을지 결정
3. App Privacy 답변을 AdMob 설정과 일치시킴
