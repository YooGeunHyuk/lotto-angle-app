# 로또될각 QA 체크리스트

최종 업데이트: 2026-05-04

## 실행 전

- `npx expo start --port 8081` 실행
- Expo Go 또는 개발 빌드에서 최신 번들 접속
- 위치/카메라 권한이 이전에 거절된 적이 있다면 iOS 설정에서 권한 상태 확인

## 내 번호

### 수동 등록

- `내 번호` 탭 진입
- 기본 회차가 최신 당첨 회차 다음 회차로 표시되는지 확인
- A 게임에 번호 6개 입력 후 저장
- 저장 목록에 `수동`, `1게임`, 저장일이 표시되는지 확인
- 같은 게임에 중복 번호 입력 시 오류 표시 확인
- 1~45 범위 밖 번호 입력 시 오류 표시 확인
- 일부 번호만 입력한 게임 저장 시 오류 표시 확인
- 저장된 번호 삭제 확인

### 당첨 결과

- 이미 당첨번호가 있는 회차로 수동 등록
- 3개 일치, 4개 일치, 5개 일치, 보너스 포함 5개 일치, 6개 일치 케이스 확인
- 당첨번호가 없는 회차는 `추첨전`으로 표시되는지 확인
- 일치 번호 테두리 표시가 깨지지 않는지 확인

### QR 스캔

- `QR 스캔` 버튼 터치
- 카메라 권한 팝업 표시 확인
- 허용 시 스캐너 화면 진입 확인
- 닫기 버튼으로 스캐너 종료 확인
- 다시 스캔 버튼이 스캔 잠금을 풀어주는지 확인
- 실제 로또 QR 스캔 시 `QR 등록` 티켓으로 저장되는지 확인
- 파싱 실패 시 QR 원문이 표시되는지 확인

## 명당

### 위치 권한

- `명당` 탭 진입 시 위치 권한 팝업 표시 확인
- 위치 허용 시 내 주변 판매점 목록이 표시되는지 확인
- 위치 거절 후 지도 안 내 위치 아이콘을 눌렀을 때 재요청 흐름 확인
- 앱 설정에서 위치 권한을 끈 뒤 다시 진입했을 때 동작 확인

### 판매점 / 내 주변 명당

- 상위 탭 `판매점`, `내 주변 명당`, `전국 명당` 위계가 구분되어 보이는지 확인
- 하위 탭 `내 주변 판매점`, `내 주변 명당`, `지도`가 보조 컨트롤처럼 보이는지 확인
- 리스트가 먼저 표시되는지 확인
- `지도` 버튼을 누르면 현재 리스트가 마커로 표시되는지 확인
- 마커 선택 시 선택한 판매점 카드가 표시되는지 확인
- 선택 카드와 리스트 카드 가로폭이 맞는지 확인

### 전국 명당

- `전국 순위`, `지역별 명당`, `최근 회차 판매점` 전환 확인
- 각 리스트의 개수와 카드 내용 확인
- `지도` 전환 시 리스트 대상이 지도에 표시되는지 확인

## 최신 데이터 자동화

- `npm run update:lotto` 실행 시 오류 없이 종료되는지 확인
- 최신 회차가 없을 때 DB가 불필요하게 바뀌지 않는지 확인
- `npm run update:lucky-stores` 실행 시 이미 최신이면 변경 없이 종료되는지 확인
- GitHub Actions에서 `Update lotto data`, `Update lucky store data` 수동 실행 가능 여부 확인

## 릴리즈 전

- Expo Go에서 광고 모듈 오류가 나지 않는지 확인
- 개발/프로덕션 빌드에서 광고 배너 표시 확인
- iOS 카메라/위치 권한 문구 확인
- Android 위치/카메라 권한 문구 확인
- 앱 아이콘, splash, adaptive icon 실제 설치 화면 확인
- 앱스토어 연령 등급에서 도박 관련 항목 확인
