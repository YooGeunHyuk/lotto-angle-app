# 말하는 텍스트뷰어 📖🔊

'텍스트뷰어' 류의 txt 리더 앱을 벤치마킹하고, 여기에 **음성으로 읽어주기(TTS)** 기능을 더한
Expo(React Native) 앱입니다.

## 주요 기능

### 텍스트 뷰어 (벤치마킹 기능)
- **txt 파일 가져오기** — 문서 선택기로 여러 파일을 한 번에 서재에 추가 (앱 저장소로 복사되어 원본 삭제와 무관하게 유지)
- **인코딩 자동 감지** — UTF-8 / CP949(EUC-KR) / UTF-16 BOM 자동 판별. 국내 txt 소설 대부분을 깨짐 없이 열 수 있음
- **서재** — 최근 읽은 순 정렬, 파일 크기·진행률·마지막 읽은 시각 표시, 길게 눌러 삭제
- **이어 읽기** — 문단 단위로 읽던 위치를 자동 저장하고 다시 열면 그 위치로 복원
- **보기 설정** — 글자 크기(12~32), 줄 간격(1.2~2.6), 배경 테마(밝게/세피아/어둡게)
- **진행률 슬라이더** — 하단 슬라이더로 원하는 위치로 즉시 이동
- **화면 항상 켜짐** — 읽는 동안 화면이 꺼지지 않음

### 음성 읽기 (TTS)
- **지금 보는 위치부터 듣기** — 하단 스피커 버튼을 누르면 현재 화면의 문단부터 한국어 음성으로 읽어줌
- **문단 길게 눌러 듣기** — 원하는 문단을 길게 누르면 그 위치부터 바로 재생
- **읽는 문단 하이라이트 + 자동 스크롤** — 지금 읽는 문단이 강조되고 화면이 따라 움직임
- **일시정지 / 이어 듣기** — 멈춘 문장부터 다시 재생
- **문장 단위 이전/다음 건너뛰기**
- **배속 조절** — 빠른 전환 버튼(x0.8 → x1 → x1.25 → x1.5 → x2) 또는 설정에서 세밀 조절, 음높이도 조절 가능

## 실행 방법

```bash
cd text-viewer-app
npm install
npx expo start
```

- Android/iOS 실기기에서는 Expo Go 또는 development build로 실행하세요.
- TTS는 기기의 시스템 음성 엔진을 사용하므로 **실기기에서 테스트**하는 것을 권장합니다.
  (Android는 설정 > 접근성 > TTS 출력에서 한국어 음성 데이터가 설치되어 있어야 합니다)

## 구조

```
app/
  index.tsx          # 서재 (파일 목록 / 가져오기 / 삭제)
  viewer/[id].tsx    # 뷰어 + TTS 컨트롤
src/
  components/SettingsSheet.tsx  # 보기·음성 설정 바텀시트
  lib/encoding.ts    # UTF-8/CP949/UTF-16 자동 감지 디코더
  lib/library.ts     # 서재 저장소 (AsyncStorage + 파일 복사)
  lib/settings.ts    # 뷰어 설정 저장
  lib/textUtils.ts   # 문단/문장 분리 (TTS 발화 단위)
  lib/theme.ts       # 읽기 테마 색상
```
